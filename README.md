- **Embedded Systems & Microcontrollers**: Experienced with ARM (STM32, Cortex-M: 407, 411, 469, 479), AVR (ATmega, ATtiny), Raspberry Pi (Raspbian, Ubuntu), and GSM/GPRS modules (SimCom SIM808, Quectel MC60).
- **Python Ecosystem**: Skilled in asyncio, NumPy, pandas, scikit-learn, matplotlib, seaborn, PyTorch, DGL, and NetworkX.
- **Backend Development**: Proficient in building robust backend solutions using C, Python, Rust, and PHP.
- **Frontend Development**: Experienced in HTML, CSS, and JavaScript, with familiarity in frameworks like Quart and Hypercorn. Currently exploring Hypercorn, Tauri, Leptos, and Slint.
- **DevOps**: Knowledgeable in containerization and orchestration with Docker and Docker Compose.
- **Databases**: Strong expertise in relational databases (MySQL, PostgreSQL) and NoSQL databases (Redis in-memory key-value store).
- **Networking Protocols**: Proficient in protocols for media content transmission and data communication.
- **Blockchain**: Solid understanding of blockchain principles and technologies.
- **Graph Neural Networks (GNNs)**: Practical experience with PyTorch, DGL, and NetworkX for GNN development.
